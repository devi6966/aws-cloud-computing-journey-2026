# AWS CLI (Command Line Interface)

## What is AWS CLI?

AWS CLI is a command-line tool that allows users to manage AWS services using commands instead of the AWS Management Console.

## Features

* Manage AWS resources from the terminal
* Supports automation and scripting
* Works on Windows, Linux, and macOS
* Supports multiple AWS services

## Benefits

* Faster than using the console
* Useful for automation
* Easy to manage resources

## Example

```bash
aws s3 ls
