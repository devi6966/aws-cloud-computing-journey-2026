# AWS History

## What is AWS?

AWS (Amazon Web Services) is the cloud computing platform of Amazon.

## History

* 2002 - Amazon started developing its internal cloud infrastructure.
* 2006 - AWS was officially launched with services like S3 and EC2.
* 2010 - AWS became widely adopted by startups and businesses.
* 2014 - AWS emerged as a leader in the cloud computing industry.
* Today - AWS is one of the world's largest cloud providers.

## Why AWS Became Popular?

* Pay as you go pricing
* Global infrastructure
* High scalability
* Large number of services

## Key Facts

* Founded by Amazon
* Launched in 2006
* Used by startups and large enterprises

## Real Life Example

AWS is like renting IT infrastructure instead of buying your own servers.
