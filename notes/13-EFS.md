# Amazon Elastic File System (EFS)

## What is Amazon EFS?

Amazon Elastic File System (EFS) is a fully managed file storage service provided by AWS. It allows multiple EC2 instances to access the same files simultaneously.

## Features

* File-level storage
* Shared storage
* Fully managed by AWS
* Automatically scales as data grows
* Supports multiple EC2 instances at the same time

## Advantages

* Easy to use
* Highly available
* Automatically scalable
* No need to manage storage infrastructure
* Suitable for shared workloads

## Use Cases

* Web applications
* Content management systems
* Shared file storage
* Big data analytics

## Real Life Example

A company has multiple EC2 instances running the same web application. All instances use a single EFS file system to access shared images, documents, and application files.

## Key Points

* EFS = Elastic File System
* Storage Type = File Storage
* Shared by multiple EC2 instances
* Automatically scales
* Managed by AWS
