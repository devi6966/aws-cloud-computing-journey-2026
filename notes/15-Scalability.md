# Scalability

## What is Scalability?

Scalability is the ability of a system to handle increasing workloads by adding or upgrading resources without affecting performance.

## Types of Scalability

* Vertical Scaling (Scale Up) – Increase the power of an existing server by adding more CPU, RAM, or storage.
* Horizontal Scaling (Scale Out) – Add more servers or instances to distribute the workload.

## Benefits

* Handles increased traffic
* Improves application performance
* Supports business growth
* Reduces downtime

## Real Life Example

An e-commerce website receives more users during a festival sale. The company adds more EC2 instances to handle the increased traffic without slowing down the website.

## Key Points

* Handles increased workload
* Vertical Scaling = Scale Up
* Horizontal Scaling = Scale Out
* Improves performance
