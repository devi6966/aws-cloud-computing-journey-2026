# Amazon Simple Storage Service (S3)

## What is Amazon S3?

Amazon Simple Storage Service (Amazon S3) is a fully managed object storage service provided by AWS to store and retrieve any amount of data from anywhere.

## Why do we need Amazon S3?

* Store unlimited data
* Backup and restore files
* Store images, videos, and documents
* Host static websites
* Archive data

## Features

* Object Storage
* Highly Scalable
* Highly Durable (11 9's)
* Secure
* High Availability
* Fully Managed

## Real Life Example

An e-commerce company stores product images, invoices, and customer-uploaded files in Amazon S3 instead of storing them on EC2 instances.

## Key Points

* Object Storage Service
* Stores data as Objects
* Unlimited Storage
* Highly Durable
* Fully Managed by AWS
