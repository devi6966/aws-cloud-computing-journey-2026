# SSH (Secure Shell)

## What is SSH?

SSH (Secure Shell) is a secure network protocol used to remotely connect to and manage Linux servers.

## Why Use SSH?

* Secure remote login
* Encrypts communication
* Executes commands remotely
* Transfers files securely

## How SSH Works in AWS

1. Launch an EC2 instance.
2. Download the Key Pair (.pem file).
3. Open Terminal.
4. Connect using the SSH command.
5. Manage the EC2 instance remotely.

## Benefits

* Secure communication
* Remote server management
* Encrypted connection
* Passwordless authentication using key pairs

## Real Life Example

A Cloud Engineer launches an EC2 Linux instance on AWS and uses SSH with a .pem key file to securely connect and configure the server.
