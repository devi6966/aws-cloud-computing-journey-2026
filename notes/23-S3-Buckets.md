# Amazon S3 Buckets

## What is an S3 Bucket?

An S3 Bucket is a container used to store objects (files) in Amazon S3. Every object in Amazon S3 must be stored inside a bucket.

## Bucket Features

* Globally unique name
* Created in a specific AWS Region
* Stores unlimited objects
* Supports access control
* Can enable Versioning and Lifecycle Policies

## Bucket Naming Rules

* Bucket name must be globally unique.
* Use only lowercase letters, numbers, hyphens (-), and periods (.).
* Bucket name must be between 3 and 63 characters.
* Bucket name cannot contain uppercase letters or spaces.

## Real Life Example

An e-commerce company creates a bucket named `company-product-images` to store all product images. The website retrieves these images directly from Amazon S3 whenever customers browse products.

## Key Points

* Bucket = Container for Objects
* Every object must be stored inside a bucket
* Bucket names are globally unique
* Created in a specific AWS Region
