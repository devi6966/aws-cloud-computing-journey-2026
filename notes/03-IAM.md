# IAM (Identity and Access Management)

## What is IAM?

IAM is an AWS service used to securely manage access to AWS resources.

## Features

* Authentication and authorization
* User and group management
* Permission control using policies
* Multi-Factor Authentication (MFA)
* Access key management

## Components

* Users
* Groups
* Policies
* Roles

## Benefits

* Improved security
* Fine-grained access control
* Centralized user management

## Real Life Example

IAM is like a company's security department that decides who can enter which room and what actions they can perform.
