# AWS Snow Family – Hands-On

This folder contains hands-on documentation and AWS Management Console exploration related to the AWS Snow Family.

## Activities Completed

- Explored the AWS Snow Family console
- Reviewed the AWS Snowball Edge job-creation workflow
- Reviewed the available job types:
  - Import into Amazon S3
  - Export from Amazon S3
  - Local compute and storage
- Reviewed the current availability status of AWS Snowball Edge

## Current Service Update

AWS Snowball Edge is no longer available to new customers. The console exploration was performed for learning and documentation purposes only.

No Snowball Edge device was ordered, and no data-transfer job was completed.
